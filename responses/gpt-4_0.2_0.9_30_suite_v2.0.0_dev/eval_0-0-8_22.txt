```javascript
function findObject(id, root_node) {
    if (root_node.id === id) {
        return root_node;
    } else if (root_node.child) {
        var result = null;
        for (var i = 0; result == null && i < root_node.child.length; i++) {
            result = findObject(id, root_node.child[i]);
        }
        return result;
    }
    return null;
}
```

This function works by first checking if the `id` of the `root_node` matches the `id` we are looking for. If it does, it returns the `root_node`. If it doesn't, it recursively calls `findObject` on each of the children of the `root_node` until it finds a match or it has checked all children. If it finds a match, it returns that. If it doesn't find a match, it returns `null`.
