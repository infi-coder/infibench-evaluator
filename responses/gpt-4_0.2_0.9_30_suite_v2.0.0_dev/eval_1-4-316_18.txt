You can sort a list in R using the `sort()` function. However, the `sort()` function doesn't work directly on lists. You need to unlist the list first, sort it, and then convert it back to a list.

Here is the function `mySort` that you asked for:

```r
mySort <- function(l) {
  # Unlist the list
  unlisted_l <- unlist(l)
  
  # Sort the unlisted data
  sorted_l <- sort(unlisted_l)
  
  # Convert back to list
  sorted_list <- as.list(sorted_l)
  
  return(sorted_list)
}
```

You can use this function to sort your list as follows:

```r
sorted_list_text_data <- mySort(list_text_data)
```

This will return your list sorted in alphabetical order.
