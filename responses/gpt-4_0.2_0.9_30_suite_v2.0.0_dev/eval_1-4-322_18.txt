You can achieve this by using the `split` function in R, which divides the data into subsets based on the levels of a factor. Here is the function `myConvert`:

```r
library(tidyverse)

myConvert <- function(my_data) {
  my_data %>%
    split(.$list_names) %>%
    map(~ pull(., list_values))
}
```

In this function, we first split the data frame `my_data` into a list of data frames based on the `list_names` column. Then, we use `map` to apply the `pull` function to each data frame in the list, extracting the `list_values` column.

Let's test this function with your data:

```r
my_data <- tibble(list_names = c("Ford", "Chevy", "Ford", "Dodge", "Dodge", "Ford"),
                  list_values = c("Ranger", "Equinox", "F150", "Caravan", "Ram", "Explorer"))

myConvert(my_data)
```

This will give you the desired output.
