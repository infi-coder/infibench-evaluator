The issue you're facing is due to the incorrect way of handling the `onClick` event in React. In React, you don't use `onclick` (all lowercase) as you would in plain HTML. Instead, you use `onClick` (camelCase) and pass it a function directly, not a string of code to be executed.

Here's the corrected version of your code:

```javascript
import React, { Component } from 'react'

class Main extends React.Component {
  constructor(props) {
    super(props);

    this.state = {
      imageURL: '',
    };

    this.handleUploadImage = this.handleUploadImage.bind(this);
  }

  handleUploadImage(ev) {
    ev.preventDefault();

    alert("Your file is being uploaded!");

    const data = new FormData();
    data.append('file', this.uploadInput.files[0]);
    data.append('filename', this.fileName.value);

    fetch('http://localhost:8000/upload', {
      method: 'POST',
      body: data,
    }).then((response) => {
      response.json().then((body) => {
        this.setState({ imageURL: `http://localhost:8000/${body.file}` });
      });
    });
  }

  render() {
    return (
      <form onSubmit={this.handleUploadImage}>
        <div>
          <input ref={(ref) => { this.uploadInput = ref; }} type="file" />
        </div>
        <div>
          <input ref={(ref) => { this.fileName = ref; }} type="text" placeholder="Enter the desired name of file" />
        </div>
        <br />
        <div>
          <button type="submit">Upload</button>
        </div>
        <img src={this.state.imageURL} alt="img" />
      </form>
    );
  }
}

export default Main;
```

In this code, I've moved the `alert` function into the `handleUploadImage` function, which is called when the form is submitted. This way, the alert will only be shown when the user clicks the "Upload" button.
