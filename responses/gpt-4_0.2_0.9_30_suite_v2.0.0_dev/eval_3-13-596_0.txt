The `myprogram-0.0.1-plain.jar` file is generated by the `bootJar` task in Spring Boot 2.5.0. If you want to disable the generation of the `*-plain.jar` file, you can configure the `bootJar` task in your `build.gradle` file to disable the `enabled` property of the `plain` classifier like this:

```groovy
bootJar {
    classifier = ''
    enabled = false
}

jar {
    enabled = true
}
```

This will only generate the `myprogram-0.0.1.jar` file and not the `myprogram-0.0.1-plain.jar` file.
