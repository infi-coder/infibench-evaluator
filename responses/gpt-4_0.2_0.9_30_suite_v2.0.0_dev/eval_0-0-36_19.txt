```javascript
import { useRouter } from "next/router";

export default function Component(){
    const router = useRouter();

    const reloadPage = () => {
        router.replace(router.asPath);
    }
}
```
In the above code, `router.replace(router.asPath)` will trigger a client-side reload in Next.js. This will cause `getInitialProps` to be called on the client-side.
