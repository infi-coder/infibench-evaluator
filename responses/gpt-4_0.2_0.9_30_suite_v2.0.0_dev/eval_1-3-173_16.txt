Sure, you can use the `calendar` module in Python to check if a year is a leap year or not. Here is a function that does what you want:

```python
import calendar

def DaysInYear(year):
    if calendar.isleap(year):
        return 366
    else:
        return 365
```

This function takes a year as an integer argument and returns the number of days in that year. The `calendar.isleap(year)` function checks if the year is a leap year or not. If it is, the function returns 366, otherwise it returns 365.
