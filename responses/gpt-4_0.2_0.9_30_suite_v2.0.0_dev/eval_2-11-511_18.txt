Here is the implementation of the function `my_2d_array`:

```c++
std::vector<std::string> my_2d_array(int N, int M) {
    std::vector<std::string> A(N * M);
    for (int i = 0; i < N; i++) {
        for (int j = 0; j < M; j++) {
            A[i*M+j] = std::to_string(i) + std::to_string(j);
        }
    }
    return A;
}
```

This function creates a 1D vector of strings `A` with size `N*M`. It then fills each element of the vector with a string representing its position in a 2D array of size `N*M`. The position is represented as a string in the format `ij`, where `i` is the row index and `j` is the column index. The function returns the filled vector.
