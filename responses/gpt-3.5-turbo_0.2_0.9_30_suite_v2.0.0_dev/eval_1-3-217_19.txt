You can use the `torch.unsqueeze()` function to add a new dimension to the tensor `d` and then concatenate it with `inps` along the last dimension using `torch.cat()` function. Here's the implementation of the `addTensor()` function:

```python
import torch

def addTensor(inps, d):
    d = torch.unsqueeze(d, dim=2)  # add a new dimension
    out = torch.cat((inps, d), dim=2)  # concatenate along the last dimension
    return out
```

You can call this function with your tensors `inps` and `d` as arguments to get the desired output tensor:

```python
out = addTensor(inps, d)
print(out.size())  # should print [64, 161, 2]
```
