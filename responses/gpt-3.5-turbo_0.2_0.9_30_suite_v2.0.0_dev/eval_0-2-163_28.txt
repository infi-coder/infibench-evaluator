You can use the following code to check if all the required fields in the `myForm` div are filled:

```javascript
document.getElementById("check").onclick = function() {
  let allAreFilled = true;
  document.querySelectorAll("#myForm [required]").forEach(function(i) {
    if (!allAreFilled) return;
    if (i.type === "radio") {
      let radioValueCheck = false;
      document.querySelectorAll(`[name=${i.name}]`).forEach(function(r) {
        if (r.checked) radioValueCheck = true;
      })
      allAreFilled = radioValueCheck;
      return;
    }
    if (!i.value) { allAreFilled = false;  return; }
  })
  if (!allAreFilled) {
    alert('Fill all the fields');
  }
};
```

This code selects all the elements inside the `myForm` div that have the `required` attribute using `querySelectorAll`. It then loops through each element and checks if it is a radio button or not. If it is a radio button, it checks if at least one of the radio buttons with the same name is checked. If it is not a radio button, it checks if the value of the element is empty. If any of the required fields are not filled, it sets `allAreFilled` to `false`. Finally, if `allAreFilled` is still `true`, it means all the required fields are filled, and the code does nothing. If `allAreFilled` is `false`, it displays an alert message asking the user to fill all the fields.
