```javascript
function findObject(id, root_node) {
  if (root_node.id === id) {
    return root_node;
  }
  for (let i = 0; i < root_node.child.length; i++) {
    const result = findObject(id, root_node.child[i]);
    if (result) {
      return result;
    }
  }
}
```

This function takes two arguments: `id` and `root_node`. It recursively searches for the object with the given `id` in the nested object tree starting from the `root_node`. If the `id` matches with the `id` of the current node, it returns the current node. Otherwise, it recursively calls itself on each child of the current node until it finds the object with the given `id`. If the object is not found, it returns `undefined`.
