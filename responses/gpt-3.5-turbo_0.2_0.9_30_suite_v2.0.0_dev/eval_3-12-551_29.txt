Either wrap your custom appbar into a `PreferredSize` widget or implement `PreferredSizeWidget`. 

In your case, you can implement `PreferredSizeWidget` by adding a `preferredSize` property to your `AppBarDesign` class. Here's how you can modify your code:

```dart
class AppBarDesign extends StatelessWidget implements PreferredSizeWidget {
  AppBarDesign(Key key, this.title) : super(key: key);

  final title;

  @override
  Widget build(BuildContext context) {
    return new AppBar(
      title: new Text(title),
    );
  }

  @override
  Size get preferredSize => new Size.fromHeight(kToolbarHeight);
}
```

By implementing `PreferredSizeWidget`, you are providing the `AppBar` widget with information about the preferred size of your custom app bar.
