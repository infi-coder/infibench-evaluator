Sure! Here's the revised code using `connect-mongo` v4:

```javascript
const session = require('express-session');
const MongoStore = require('connect-mongo').default;

...

app.use(session({
    secret: 'story book',
    resave: false,
    saveUninitialized: false,
    store: MongoStore.create({ mongoUrl: process.env.MONGODB_URI })
}));
```

Note that in `connect-mongo` v4, you need to use `MongoStore.create()` instead of `new MongoStore()`. Also, you need to pass the `mongoUrl` option instead of `mongooseConnection`.
