Yes, you can use an object to store all the values and update them using a single `changeHandler` function. Here's an example:

```javascript
const [allValues, setAllValues] = useState({
   mobile: '',
   username: '',
   email: '',
   password: '',
   confirmPassword: ''
});

const changeHandler = e => {
   const { name, value } = e.target;
   setAllValues(prevState => ({ ...prevState, [name]: value }));
};

return (
   <input type="text"
       className="form-control"
       id="mobile"
       name="mobile"
       placeholder="Enter a valid mobile number"
       value={allValues.mobile}
       onChange={changeHandler}
   />
   // ...
)
```

In the `changeHandler` function, we use destructuring to get the `name` and `value` properties from the event target. Then, we use the spread operator to create a new object with all the previous values and update the one that matches the `name` property. Finally, we pass this new object to the `setAllValues` function to update the state.

Note that we also added the `value` property to the input element to display the current value from the state.
