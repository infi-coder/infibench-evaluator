You need to add a `go_package` option to your `.proto` file. The `go_package` option specifies the Go package name for the generated code. Here's an example of how to add the `go_package` option to your `.proto` file:

```go
syntax="proto3";

package main;

option go_package = "github.com/your-username/your-repo/your-package-path";

message Person {
      string name = 1;
      int32 age = 2;
}
```

Replace `github.com/your-username/your-repo/your-package-path` with the actual import path of your Go package. Then, run the `protoc` command again with the `--go_out` flag and the `M` argument:

```bash
protoc --go_out=plugins=grpc,M<your-package-path>=github.com/your-username/your-repo/your-package-path:. simple.proto
```

Replace `<your-package-path>` with the actual package path of your Go package. This should generate the Go code for your `.proto` file.
