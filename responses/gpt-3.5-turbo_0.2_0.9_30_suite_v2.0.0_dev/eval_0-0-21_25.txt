To copy the text from the `v-text-field` to the clipboard, you can use the `navigator.clipboard.writeText()` method. Here's the updated code:

```html
<template>
  <div id="app">
    <v-app id="inspire">
      <v-container>
        <v-text-field v-model="text1"></v-text-field>
        <v-btn @click="copyText">copy</v-btn>
      </v-container>
    </v-app>
  </div>
</template>

<script>
  new Vue({
    el: '#app',
    vuetify: new Vuetify(),
    data() {
      return {
        text1: 'lorem ipsum 123'
      }
    },
    methods: {
      copyText() {
        navigator.clipboard.writeText(this.text1)
          .then(() => {
            console.log('Text copied to clipboard');
          })
          .catch((error) => {
            console.error('Error copying text: ', error);
          });
      }
    }
  })
</script>
```

In the `copyText` method, we use the `navigator.clipboard.writeText()` method to write the text from the `text1` data property to the clipboard. We also add a `then()` method to log a message to the console when the text is successfully copied, and a `catch()` method to log an error message if there's an error copying the text.
