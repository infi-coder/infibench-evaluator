You can use `ElevatedButton`, `TextButton` and `OutlinedButton` that are not deprecated button themes.
You can change the `shape` property which placed in the `ButtonStyle` property. Set the `shape` property to `RoundedRectangleBorder` and set the `borderRadius` property to the desired value. For example:

```
ElevatedButton(
  onPressed: () {},
  child: Text('Button'),
  style: ButtonStyle(
    shape: MaterialStateProperty.all<RoundedRectangleBorder>(
      RoundedRectangleBorder(
        borderRadius: BorderRadius.circular(18.0),
      ),
    ),
  ),
),
```
