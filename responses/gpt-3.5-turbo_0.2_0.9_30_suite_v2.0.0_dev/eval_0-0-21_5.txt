To copy the text from the `v-text-field` component to the clipboard when the button is clicked, you can use the `navigator.clipboard.writeText()` method. Here's the updated code:

```html
<template>
  <div id="app">
    <v-app id="inspire">
      <v-container>
        <v-text-field v-model="text1"></v-text-field>
        <v-btn @click="copyText">copy</v-btn>
      </v-container>
    </v-app>
  </div>
</template>

<script>
  new Vue({
    el: '#app',
    vuetify: new Vuetify(),
    data() {
      return {
        text1: 'lorem ipsum 123'
      }
    },
    methods: {
      copyText() {
        navigator.clipboard.writeText(this.text1)
          .then(() => {
            console.log('Text copied to clipboard');
          })
          .catch((error) => {
            console.error('Error copying text: ', error);
          });
      }
    }
  })
</script>
```

In the `copyText` method, we use the `navigator.clipboard.writeText()` method to write the value of `text1` to the clipboard. The `then()` method is called if the text is successfully copied, and the `catch()` method is called if there is an error. In this example, we just log the success or error message to the console.
